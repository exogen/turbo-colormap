turbo-colormap

A TypeScript library for working with Turbo, an improved rainbow colormap for visualization, often used for depth maps. In addition to the library, you can also use this demo page to perform quick color and image conversions. Note: this library generates more faithful values than the d3 approximation of Turbo found in d3-scale-chromatic.

See the README on GitHub.

npm install turbo-colormap

nearest color

Snap arbitrary RGB colors to the quantized Turbo palette. This is mostly useful for visualizations exported with an approximation of Turbo or a lossy format (like JPEG).

The nearest color is determined by simple Euclidian distance using a k-d tree.

image converter

Some tools expect grayscale depth maps. Load images into the area below to convert between grayscale and Turbo images. Conversion is done client-side; all data stays in your browser.
